# Quick Setup ðŸš€

Follow these steps to create your interactive portfolio in just 5 minutes.

## Step 1: Create New Project

```bash
# Create a new React project
npx create-react-app my-portfolio
cd my-portfolio

# Install AdzyJS
npm install adzyjs
```

## Step 2: Basic Portfolio Structure

Replace the contents of `src/App.js`:

```jsx
import { Player } from 'adzyjs';
import './App.css';

function App() {
  return (
    <div className="portfolio">
      {/* Hero Section */}
      <section className="hero">
        <div className="hero-content">
          <h1>Hi, I'm [Your Name]</h1>
          <p>Full Stack Developer & Creative Coder</p>
          <button className="cta-button">View My Work</button>
        </div>
        <div className="hero-character">
          <Player 
            skinUrl="https://example.com/your-skin.png"
            animation="wave"
            width={250}
            height={350}
          />
        </div>
      </section>

      {/* Skills Section */}
      <section className="skills">
        <h2>Skills</h2>
        <div className="skills-grid">
          <div className="skill-item">
            <Player 
              skinUrl="https://example.com/js-dev.png"
              animation="idle"
              width={150}
              height={200}
            />
            <h3>JavaScript</h3>
          </div>
          <div className="skill-item">
            <Player 
              skinUrl="https://example.com/react-dev.png"
              animation="idle"
              width={150}
              height={200}
            />
            <h3>React</h3>
          </div>
        </div>
      </section>

      {/* Contact Section */}
      <section className="contact">
        <div className="contact-content">
          <h2>Let's Work Together</h2>
          <p>Ready to bring your ideas to life?</p>
          <button className="contact-button">Get In Touch</button>
        </div>
        <div className="contact-character">
          <Player 
            skinUrl="https://example.com/contact-character.png"
            animation="thumbsup"
            width={200}
            height={280}
          />
        </div>
      </section>
    </div>
  );
}

export default App;
```

## Step 4: Get Your Minecraft Skin

1. **Use Your Own Skin**: Upload your Minecraft skin to a CDN or use services like imgur
2. **Use Default Skins**: 
   - Steve: `https://textures.minecraft.net/texture/...`
   - Alex: `https://textures.minecraft.net/texture/...`
3. **Create Custom Skins**: Use tools like [MCSkin3D](https://www.mcskin3d.net/) or [NovaSkin](https://www.minecraftskins.com/skin-editor)

## Step 5: Run Your Portfolio

```bash
npm start
```

Visit `http://localhost:3000` to see your portfolio!

## Step 6: Customize Your Content

1. **Update Personal Info**: Replace `[Your Name]` and job title
2. **Add Your Skills**: Update the skills section with your technologies
3. **Update Skin URLs**: Replace example URLs with your actual skin URLs
4. **Modify Colors**: Adjust the CSS colors to match your brand

## Quick Deployment

Deploy to Netlify in seconds:

```bash
npm run build
npx netlify-cli deploy --prod --dir=build
```
